# Periodicity Analysis Lab

## Pitch Detection Lab

Compare periodicity analysis methods:

- FFT
- Autocorrelation
- YIN (CMNDF)
- bedcmm

Features
- Waveform visualization
- Frame-by-frame analysis
- Pitch track comparison
